Smart Garage Pi

An Alexa skill to control smart garages enabled via a Raspberry Pi

This skill is to enable the controlling of a regular garage that has been made "smart" using a Raspberry Pi along with a Camera module, a two-channel relay, a magnetic garage door switch and two different open source libraries.
