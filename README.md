# Smart Garage Pi

> An Alexa skill to control smart garages enabled via a Raspberry Pi

This skill is to enable the controlling of a regular garage that has been made "smart" using a Raspberry Pi along with a Camera module, a two-channel relay (https://smile.amazon.com/gp/product/B0057OC6D8/), a magnetic garage door switch (https://smile.amazon.com/gp/product/B00CAIVNM4/) and two different open source libraries (https://www.richlynch.com/2013/07/27/pi_garage_alert_1/ and https://coderwall.com/p/jsd5mw/raspberry-pi-garage-door-opener-with-garagepi).
